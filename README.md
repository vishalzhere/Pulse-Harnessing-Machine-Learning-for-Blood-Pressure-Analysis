# Pulse-Harnessing-Machine-Learning-for-Blood-Pressure-Analysis
## Project Overview
Predictive Pulse is a machine learning-based web application that predicts blood pressure risk using patient health parameters. The system analyzes clinical data and provides predictions to help in early detection and monitoring of hypertension.

Predictive Pulse is a Machine Learning based system designed to predict blood pressure conditions using patient health parameters. The project helps in early detection and monitoring of hypertension by analyzing medical and lifestyle data.

This project is developed as part of the Skill Wallet AI & Machine Learning Program.

## How the System Works

1. User enters patient health details.
2. Data is preprocessed and scaled.
3. Machine learning model analyzes the data.
4. System predicts blood pressure condition.
5. Prediction result is displayed.
